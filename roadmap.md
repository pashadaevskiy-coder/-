# Roadmap: Лендинг для независимой экспертизы инженерных проектов

## Обзор проекта
Создание современного, минималистичного и строгого лендинга для B2B услуг в сфере инженерии и строительства. Одностраничный сайт с адаптивным дизайном, формами заявки и функционалом плавного скролла.

**Целевой стек:** Чистый HTML5, CSS3, Vanilla JavaScript (без фреймворков)

**Основные файлы:**
- `index.html` – структура страницы
- `style.css` – стили
- `script.js` – интерактивность

## Этап 2: Базовая HTML-структура

### [ ] 2.2. Секция 1: Главный экран (Hero)

- Текст: "Независимый инженер с опытом более 20 лет проверяет проект и выявляет ошибки на ранней стадии"

## Этап 3: Стилизация (CSS)

### [ ] 3.1. Базовые стили и сброс
- Normalize или reset CSS
- Задать шрифты (системные, например, Inter или Roboto)

### [ ] 3.2. Общая сетка и контейнеры
- Максимальная ширина контейнера (например, 1200px)
- Отступы, внутренние padding
- Использование flex/grid для выравнивания

### [ ] 3.3. Стилизация каждой секции
- Hero: крупный заголовок, фон, кнопка с hover-эффектом
- Блок проблем: карточки или список с иконками
- Решение: акцентный блок
- О специалисте: сетка из 4 пунктов
- Преимущества: иконки с текстом
- Формы: стилизация полей ввода, кнопок
- Контакты: расположение иконок соцсетей

### [ ] 3.4. Адаптивность (мобильная версия)
- Медиазапросы для экранов ≤768px
- Изменение сетки на одноколоночную
- Уменьшение размеров шрифтов, отступов
- Адаптация форм и кнопок

### [ ] 3.5. Детали и эффекты
- Hover-эффекты для кнопок, ссылок, карточек
- Плавные переходы (transition)
- Тени, границы, скругления
- Иконки (использовать Font Awesome или SVG)
